Aplicación web para gestión de restaurante

Bistro FDI es un proyecto de aplicación web orientado a la gestión de un restaurante. Su objetivo es facilitar tareas habituales como consultar la carta, gestionar reservas y coordinar pedidos entre sala y cocina. La aplicación está pensada para distintos tipos de usuarios (cliente, camarero, cocina y gerente), cada uno con funcionalidades adaptadas a su rol. En esta práctica se presenta una primera versión estática con la descripción del proyecto, los bocetos, la planificación y la información del equipo.
