Planificación del Proyecto


Desglose de Tareas a Realizar

El desarrollo del proyecto Bistro FDI se ha dividido en tres fases principales, alineadas con las diferentes entregas de la práctica.

Fase 1: Prototipado y Funcionalidades Núcleo (Entrega 2)

Objetivo: Implementación de la lógica de servidor y estructura HTML básica.

  • Base de Datos: Modelado E-R (Usuarios, Productos, Pedidos).
  • Funcionalidad 0 (Usuarios): Registro, login y gestión de roles (Cliente, Camarero, Cocinero, Gerente).
  • Funcionalidad 1 (Productos): CRUD de productos y categorías (Vista Gerente).
  • Funcionalidad 2 (Pedidos): Carrito de compra, confirmación y gestión de estados básicos.
  • Funcionalidad 3 (Cocina): Vista de preparación.

Fase 2: Estilos y Ampliación (Entrega 3)

Objetivo: Mejora visual e implementación de lógica de negocio compleja.

  • Diseño Visual: Aplicación de CSS y diseño responsivo para móviles/tablets.
  • Funcionalidad 4 (Ofertas): Gestión de packs y descuentos.
  • Despliegue: Configuración del servidor VPS.

Fase 3: Finalización (Entrega Final)

Objetivo: Aplicación completa y depurada.

  • Funcionalidad 5 (Recompensas): Sistema de puntos "BistroCoins".
  • Testing: Pruebas de flujo completo y corrección de errores.
